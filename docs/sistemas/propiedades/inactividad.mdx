# Inactividad y Liberación Automática

Si un propietario deja de entrar a su propiedad, el sistema la **libera automáticamente**. Los plazos dependen del tipo:

| Tipo de propiedad | Días sin actividad |
| --------------- | ---------------- |
| Casas compradas | 60 días |
| Casas alquiladas | 14 días |
| Negocios | 14 días |
| Almacenes | 90 días |
| Negocios sucios | 7 días |

Cuando una propiedad se libera: se despide a los empleados, se vacía la caja, se expulsan a los jugadores, se resetean las llaves y se limpian los inventarios.
