# Incendios y Extintores

Los negocios pueden **sufrir incendios** de forma aleatoria cuando hay bomberos en servicio. Si el local **no tiene extintores** en ese momento, se genera un incendio que daña el inventario, apaga la emisora, activa la alarma y avisa a bomberos y policías. Tras un incendio, el negocio queda con un **cooldown** antes de poder incendiarse otra vez.

Cada negocio puede almacenar hasta **10 extintores llenos**. Los extintores se consumen cuando el sistema los usa para prevenir un incendio.

## Comandos

- `/extintores sacar` — retira un extintor del almacén del negocio.
- `/extintores rellenar` — guarda un extintor lleno de vuelta al almacén.
